# AI Study Assistant CLI

A terminal-based AI study assistant built using Python and OpenRouter API.

## Features
- Ask study-related questions
- AI-generated responses
- Chat history saving
- Built-in commands

## Commands
- `/help` → Show all commands
- `/history` → View saved history
- `/clear` → Clear history
- `/exit` → Exit the program

## Technologies Used
- Python
- Requests
- OpenRouter API
- File Handling

## How to Run

Install requests:

pip install requests

Run the program:

python main.py
